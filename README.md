# StockFlow - Inventory & Route Management System

A comprehensive inventory management system with AI-powered route planning and material estimation.

## Features

- **Inventory Management**: Track stock levels, costs, and suppliers
- **Stock Movements**: Monitor all inventory changes with full audit trail
- **Route Planner**: Plot and analyze fiber optic cable routes
- **AI Analysis**: Get detailed project reports using Ollama (Qwen2.5 3B)
- **Cost Estimation**: Automatic material and labor cost calculations
- **Excel Export**: Download detailed reports and material schedules

## Technology Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: Firebase Realtime Database
- **AI**: Ollama with Qwen2.5 3B
- **Mapping**: Leaflet.js
- **Charts**: Chart.js
- **Excel**: OpenPyXL, XLSX

## Installation

### Prerequisites

- Python 3.8+
- Node.js (for frontend development)
- Ollama (for AI features)
